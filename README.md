# zomato-eda-analysis
Project Overview This project performs Exploratory Data Analysis (EDA) on the Zomato Restaurant Dataset using Python. The goal of this project is to analyze restaurant trends, customer ratings, cuisines, pricing, and online delivery services.

Technologies Used

Python
Pandas
NumPy

Key Insights

Major cities contain the highest number of restaurants.
Online delivery restaurants generally receive better ratings.
Medium-priced restaurants are more common.
Customer satisfaction depends more on quality and service than price.
Dataset Zomato Restaurant Dataset from Kaggle.
